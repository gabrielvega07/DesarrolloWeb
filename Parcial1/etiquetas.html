# Etiqueta HTML Descripción
1 <!DOCTYPE html> Define el tipo de documento como HTML5.
2 <html> Define el elemento raíz del documento HTML.
3 <head> Contiene informacion de encabezado sobre el documento.
4 <meta> Especifica metadatos sobre el documento, como la codificación de caracteres.
5 <title> Establece el título del documento, que se muestra en la barra de título o pestaña del navegador.
6 <link> Enlaza un recurso externo, como una hoja de estilos CSS, con el documento HTML.
7 <body> Define el cuerpo del documento, que contiene el contenido visible en el navegador.
8 <h1> Define un encabezado de nivel 1, que generalmente se usa para el título principal de la página.
9 <div> Define un contenedor de bloque genérico que se utiliza para agrupar elementos.
10 <p> Define un párrafo de texto.
11 <img> Inserta una imagen en la página.
12 <ul> Define una lista no ordenada, que se muestra con viñetas.
13 <li> Define un elemento de lista en una lista no ordenada o una lista ordenada.
14 <a> Define un enlace o hipervínculo a otra página o recurso.
15 <h2> Define un encabezado de nivel 2, que generalmente se usa para subtítulos.
16 <h3> Define un encabezado de nivel 3, que generalmente se usa para sub-subtítulos.
17 <form> Define un formulario para enviar datos al servidor web.
18 <input> Define un campo de entrada en un formulario.
19 <script> Define un script o código JavaScript que se ejecutará en la página.
20 <button> Define un botón interactivo.
21 <iframe> Inserta un marco en línea que permite mostrar otra página web dentro de la página actual.
